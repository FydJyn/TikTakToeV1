# TikTakToeV1
# Крестики-нолики — версия 1

Первая версия лабораторной работы по C# Windows Forms.

## Возможности
- обычная игра 3x3 для двух игроков;
- поочередные ходы X и O;
- определение победителя;
- определение ничьей;
- кнопка перезапуска игры.

## Как запустить
1. Открыть файл `TicTacToeV1.sln` в Visual Studio 2022 или новее.
2. Убедиться, что установлен .NET 8 SDK и workload для Windows Forms.
3. Нажать `Start`.
